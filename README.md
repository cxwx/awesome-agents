## CLI Agents

| Name | Description |
|------|-------------|
| **[Aider](https://github.com/paul-gauthier/aider)** | AI pair programming in your terminal. Supports multiple LLM providers. |
| **[Goose](https://github.com/block/goose)** | Open-source AI agent for automating engineering tasks. |
| **[Kilo-CLI](https://github.com/kilocode/kilo-cli)** | CLI-based AI coding assistant. |
| **[Kimi-CLI](https://github.com/MoonshotAI/kimi-cli)** | CLI client for Kimi AI by Moonshot AI. |
